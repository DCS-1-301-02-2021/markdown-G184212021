# 何を学ぶのか:eyes:

**実社会に役立つ総合的能力の修得**
- 基礎スキル
- プログラミング
- コース選択による専門的知識
- 卒業研究によって、主体性、課題に対する洞察力、解決のためのコミュニケーション能力等

# コース
**基本のコース**
1. システムエンジニアリングコース
2. インターネット＆セキュリティコース
3. 知能情報コース

**コラボレーションコース**

1. ビジュアルコンピューティングコース
2. Webデザインコース
